# pokemon-api-explorer
A simple Pokémon search app built with JavaScript using the PokeAPI. You can search any Pokémon by name and view its image, type, and base stats. This project helped me practice working with APIs, async/await, and DOM manipulation in a real mini web app.

This project was built as a hands-on way to practice working with APIs and improve my JavaScript skills, especially async/await, DOM manipulation, and handling user input.

It’s not a perfect or final version — just a learning project that helped me understand how real web apps fetch and display dynamic data.

#  Features
Search Pokémon by name
Display Pokémon image
Show type(s)
Show base stats using meters
Handles invalid inputs gracefully

# Built With
HTML
CSS
JavaScript
PokeAPI


#  What I learned
How APIs work in real projects
Using fetch with async/await
Updating the DOM dynamically
Handling errors in API calls
Structuring a small frontend project
